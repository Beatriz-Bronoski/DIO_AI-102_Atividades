# DIO_AI-102_Atividades
Repositório reservado para as atividades contidas no curso de IA-102 promovido na plataforma da DIO

### Tradutor de artigos técnicos
Este artigo irá trazer o passo a passo para utilizar o tradutor de artigos técnicos do Azure ( Azure AI ).
_____________________________________________________________________________________________________________________________________________________________________________________________

### 1° Abra o Portal do Azure
  Crie uma conta gratuita ou de estudante para acessar aos serviços, caso sua universidade disponibilize e-mails para estudantes. 
### 2° Acesse o grupo de recursos do Azure e crie um novo grupo
### 3° Busque, dentro desse grupo de recursos, o produto "Tradutores"
  Busque no portal por serviços de IA e encontre por "Tradutores" ou "Translator". Neste caso, o valor é cobrado por uso. 
### 4° Busque, dentro do portal, por "Azure Open AI" e crie um novo serviço
  Dentro desse serviço, você deve abrir o Azure AI Studio, e selecionar um modelo de sua preferência.
